Трапезна для студентів УКУ - другий дім. Ми, як студенти, не завжди харчуємося правильно через брак часу і заклопотаність. Саме тому ми вирішили створити веб-застосунок, який би спростив життя кожного з нас і допоміг легко підібрати собі корисний раціон та зекономити час. 

## Мета нашого проєкту
Проєкт створений студентами УКУ як інструмент для обчислення нутрієнтів та їх правильного розподілу, а також як засіб для відстежування свого щоденного раціону. Як-тільки но наш проєкт було задумано, нашою метою було створити функціонал, який працював би тільки для Трапезної. Але в процесі ми зробили наш веб-застосунок сумісним з будь-яким меню при виконанні декількох умов(правильний формат подання меню та його відповідні налаштування по замовчуванню).

## Основний алгоритм
Веб-застосунок базується на алгоритмі, який на основі певного меню підбирає набори страв і виділяє з них ті, які якнайточніше відповідають вимогам користувача(надані нутрієнти). Для збільшення точності підбору, страви урізноманітнюються порціями. 

## Навігація для веб-застосунку
У нашому веб-застосунку є гостьова версія(для незареєстрованих користувачів) та версія для зареєстрованих користувачів.

Незареєстровані користувачі мають доступ лише до загальнодоступних інструментів(перегляд вкладки Menu та користування Calculator).
Зареєстровані користувачі мають доступ до ширшого функціоналу, як-от планування свого меню, збереження обчислень, підбирання особистого плану, регулювання своїх даних, налаштування алгоритмів підбору їжі під себе і т.д

## Формат меню
Меню - набір категорій(у нашому випадку це Breakfasts, Second Meals, Salads, Garnirs, Soups), в кожній з яких міститься набір страв з даними нутрієнтами(калорії, білки, вуглеводи, жири) для однієї порції.

### Незареєстровані користувачі
* доступ лише до вкладки Menu та калькулятора

* Калькулятор - інструмент для одноразового підбору страв для одного прийому їжі. Показує 5 найбільш задовільних варіантів і не зберігає нікуди дані.
* Меню - усіх страв з вказаними нутрієнтами для однієї порції(обсяг однієї порції може змінюватися в залежності від меню). 

Перш за все, користувач повинен надати кількість білків, жирів і вуглеводів для прийому їжі, який він хоче спожити.

### Зареєстровані користувачі
Після реєстрації та успішної автентифікації, користувача переносить на сторінку Main, яка є персональною для кожного. На ній користувач має змогу створити власний раціон і, відповідно, його змінювати(додавати та видаляти прийоми їжі). Для зручності, користувач може скористатися кнопкою Daily Distribution, яка додасть прийоми їжі в залежності від налаштувань. Прийоми їжі також можна додавати поштучно, вводячи для кожного вручну нутрієнти.

Процес підбирання страв є аналогічним до процесу, описаного вище(Calculator page). Після генерації або створення кожен прийом їжі має закріплену кількість нутрієнтів за собою. Після генерації страв, всі отримані варіанти закріплюються за прийомом їжі, і одному з них надається статус "Обрано". Користувач може міняти обрану страву. Калорії для кожного прийому їжі підтягуюються з відповідно обраної страви.
Для зручності на екрані відображається шкала, яка допомагає відслідковувати скільки калорій спожито.

### Акаунт користувача

На основі наданої користувачем особистої інформації, ми генеруємо підбір нутрієнтів на весь день, розподілений між прийомами їжі(кількість прийомів їжі користувач може міняти).

<img width="500" alt="image" src="https://user-images.githubusercontent.com/116542057/230797073-b643911c-1c43-4835-94a9-02e0437ce69b.png">

Користувач також може змінювати особисту інформацію, відповідно його підбір нутрієнтів теж буде змінюватися.
Якщо користувача не задовільняє авто-згенерований підбір нутрієнтів, він може підібрати його власноруч, перейшовши за Personal Plan -> Mannual, і обравши бажану кількість нутрієнтів на день. 

Користувачу надана можливість регулювати алгоритм підбору страв, а саме: налаштувати, які категорії страв не повинні повторюватися, які кількості порцій для них дозволені, скільки варіантів підбору користувачу буде показувати і максимальну кількість страв в одному прийомі їжі.
